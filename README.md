* Email: ewan@axisfx.design

* Current Version: 1.0.0

## Supported Objects/Tags

* Alembic Object
* Point Cache Tag
* Mograph Cache Tag

## Installation

* Click the green 'Code' button
* Click 'Download ZIP'
* Extract to the Cinema 4D R2X/Plugins folder
* Restart Cinema 4D
